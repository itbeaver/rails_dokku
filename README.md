# RailsDokku

Rails dokku is tasks collection for (dokku)[https://github.com/dokku/dokku]

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rails_dokku'
```

And then execute:

    $ rake -T

Or install it yourself as:

    $ gem install rails_dokku

## Usage

Please look at rake -T.
